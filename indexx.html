Get To Know!

Mini Car Mobile Remote ControlWhat is That?

Arduino Bluetooth RC or what we name MCMRC (Mini Car Mobile Remote Control) is a robot that can follow commands like the usual remote control car, the difference is that this robot uses bluetooth as a means of communication with android. The application used is Arduino Bluetooth RC Car which can be downloaded for free on PlayStore. The bluetooth module on this RC uses HC-05, L298N motor driver, and Arduino Uno as the main controller.

Our Goals

Building an Arduino-based
RC Vehicle

The aim of this project is to create an Arduino-based remote control car that can be controlled via Bluetooth communication using an application on a smartphone. This project is designed to introduce the concepts of microcontrollers, wireless communication, and programming, and provide practical experience in developing functional and innovative electronic devices.

Read More
Our Benefits

Flexibility and Customization
& Development of Creativity and Problem Solving

The project provides both educational and practical benefits, such as improving technical understanding of the operation of electronic components, particularly Arduino and Bluetooth modules. In addition, the benefits include cost savings by self-assembling the RC car as well as encouraging creativity in solving technical challenges, making it a fun and rewarding learning platform.

Our Achievements

Optimal Control Program
& Modular System

The main achievement of this project was to create a functional RC car that can move on command from a smartphone app. The project also demonstrated the integration capabilities between hardware and software, resulting in a modular prototype that can be further developed, such as by adding automation features or advanced sensors.

2. Assemble the Car

Assemble the car by installing the motor in the appropriate position on the wooden board. Make sure the motor is firmly attached to avoid damage during use.

3. Component Connection

Connect all components such as Arduino, Bluetooth module, battery power connection, motor driver and also Dc motor correctly and precisely.

4. Arduino Programming

Write a program to control the car based on commands from the Bluetooth module. This program will read the signal from the smartphone and drive the motor according to the command received.
